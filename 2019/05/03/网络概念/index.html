客户端与服务器

本质上都是计算机，只不过样子不同，配置不同，应用场景不同（安装的应用软件不同）

  1. 客户端主要用于普通上网用户
  2. 服务器主要给上网用户提供后台服务

网络相关概念

  1. IP地址（唯一的确定互联网上的一台计算机）
  2. 域名 IP地址的别名，方便记忆
  3. DNS 用于维护IP地址与域名的关系
  4. 端口 用来确定计算机上的网络应用程序

通信双方约定的规则

  1. http/https 超为本传输协议
  2. ftp 文件传输协议
  3. smpt/pop3 邮件收发协议
  4. ……
